## 2020 OS 实习第二阶段目标
车春池  
2020/08/07  
### 目标
熟悉 zCore，实现将 zCore 的一部分在 RISCV 架构上的 QEMU 上运行。  
### 大致思路
zCore 中与硬件交互的模块是 HAL 硬件抽象层，将 zCore 移植到 RISCV 上需要将这部分进行修改使其兼容 RISCV 架构。目前的大概思路是先从 HAL 中的一部分 API 着手，然后再一步步增加对 HAL 中的 API 的 RISCV 下的重构。最终达到能在 RISCV 架构下的 QEMU 中运行 zCore 的核心部分。  
### 预测会遇到的困难：  
+ 需要时间学习 zCore
+ 对于一些基础的系统知识，需要补缺
+ 需要熟悉 QEMU
+ Debug 会比较困难
